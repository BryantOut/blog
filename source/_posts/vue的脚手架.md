# 描述

![](/mdImg/vue-cli2.png)

vue-cli 是官方的一个脚手架工具，所谓脚手架呢就是一个架子，什么是架子呢？项目结构的架子，里面有一些基本的结构配置。利用 vue-cli 呢，我们可以生成这样的一个脚手架，所以呢它就被称为 vue 脚手架工具。

![](/mdImg/vue-cli3.png)

**下载工具**

```html
npm install vue-cli -g
```

**通过上面的工具下载模板**

```html
vue init webpack admin
```
